(DOM Testing – Day 3)

This module demonstrates manual DOM testing in vanilla JavaScript using
'console.assert'  to verify DOM behavior without user interaction.


( What This Covers )

* Creating fake DOM elements
* Testing DOM text and style updates
* Error and success UI scenarios
* Isolated DOM logic testing



## Files

- 'dom.js' – DOM update logic
- 'dom-test.js' – Manual DOM tests
- 'index.html' – Loads logic and tests
